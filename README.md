# CVD

This repository provides the processed datasets used in the study entitled *Association of Social Determinants With Cardiovascular Disease Prevalence and Healthcare Costs: A Comparative Analysis of Transformer and XGBoost*.

## Repository Contents

- `carecost.csv`: processed county-level dataset used for the analysis of total healthcare cost per capita.
- `inpatientcost.csv`: processed county-level dataset used for the analysis of inpatient healthcare cost per capita.
- `outpatientcost.csv`: processed county-level dataset used for the analysis of outpatient healthcare cost per capita.
- `heartdiseaseinsured.csv`: processed county-level dataset used for the analysis of cardiovascular disease prevalence.

## Dataset Description

These processed county-level datasets were constructed from publicly available data sources cited in the manuscript. Each dataset contains social determinant predictors and one outcome variable for the corresponding analysis.

### Main Predictors

The datasets include predictors representing:

- clinical and behavioral risk factors
- socioeconomic indicators
- demographic composition
- social and structural indicators
- rural–urban category indicators

### Data Sources

The processed datasets were constructed from publicly available data sources cited in the manuscript, including:

- CDC Interactive Atlas of Heart Disease and Stroke
- Centers for Medicare and Medicaid Services (CMS)
- American Community Survey (ACS)
- PolicyMap
- CDC Social Vulnerability Index (SVI)

## Notes

This repository contains the processed datasets used for the study. It does not include raw source files downloaded from the original providers. Users should consult the original data providers for source data access, update history, and any applicable use conditions. Detailed descriptions of data harmonization and variable construction are provided in the manuscript.
